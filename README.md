# demo_go
GO RESful API demo


Warnings:

 - GO must be installed <link>https://go.dev/dl/</link>
 
Execution: 

 - Get dependencies:
    <code>go get .</code>
 
 - Run Gin web server:
    <code>go run .</code>
 
 BugFix:
 
 If you are running the demon on a Mac, you have to execute:
 <code>go get -u golang.org/x/sys</code>
